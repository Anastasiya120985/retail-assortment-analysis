# Анализ товарного ассортимента магазина

## Описание проекта

### Аналитическая задача
Провести анализ товарного ассортимента и определить какие товары входят в основной и дополнительный ассортимент.

**Дополнительный ассортимент** — товары, которые можно предлагать как сопутствующие или располагать в прикассовой зоне.

### Описание данных
| Признак | Комментарий |
|---------|-------------|
| `date` | Дата заказа |
| `customer_id` | Идентификатор покупателя |
| `order_id` | Идентификатор заказа |
| `product` | Наименование товара |
| `quantity` | Количество товара в заказе |
| `price` | Цена товара |

## Структура проекта

### 1. Загрузка данных
- Подключение необходимых библиотек (pandas, numpy, matplotlib, seaborn, datetime)
- Чтение данных из CSV-файла
- Предварительный анализ данных:
  - Проверка структуры данных (6737 записей, 6 столбцов)
  - Проверка пропущенных значений (отсутствуют)
  - Анализ типов данных

### 2. Подготовка данных
- Корректировка типов данных (преобразование `date` в формат datetime)
- Поиск и обработка дублирующихся записей (дубли отсутствуют)
- Добавление новых столбцов:
  - `revenue` - общая выручка по каждому товару
  - `date_order` - день заказа
  - `month_order` - месяц заказа
- Анализ аномальных значений в количественных признаках (`quantity`, `price`)

### 3. Визуализация данных
- Построение гистограмм распределений для `quantity` и `price`
- Создание boxplot'ов для анализа выбросов
- Добавление вертикальных линий для среднего и медианы

## Ключевые особенности данных

### Количественные характеристики:
- **Количество товаров (quantity):**
  - Среднее: 2.50
